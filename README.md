## 📂 Projects

### ✅ Task 1 - Movie Genre Classification

A Machine Learning model that predicts the genre of a movie based on its plot description using Natural Language Processing (NLP).

### Technologies Used
- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorizer
- Multinomial Naive Bayes
- Logistic Regression
- Matplotlib
- Seaborn
- Joblib

### Features
- Dataset preprocessing
- TF-IDF text vectorization
- Naive Bayes classifier
- Logistic Regression classifier
- Accuracy comparison
- Classification Report
- Confusion Matrix
- Model Saving
- Interactive Movie Genre Prediction

**Best Model Accuracy:** **58.84%**
